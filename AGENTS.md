# AGENTS 规则

## 规则列表

### 规则 1: 第一轮对话必须阅读 README

在每次新的对话（Session）开始时，Agent 的第一轮响应必须先阅读当前项目的 `README.md` 文件（如果存在），再回答用户的问题或执行后续任务。

- **目的**: 确保 Agent 充分理解项目的背景、目标、技术栈、目录结构和约定，从而提供更准确、符合项目上下文的回答。
- **触发条件**: 新对话的第一轮用户输入。
- **要求**:
  1. 检查项目根目录下是否存在 `README.md`。
  2. 如果存在，先读取 `README.md` 的内容。
  3. 如果不存在，跳过此规则，正常响应用户。
  4. 阅读完成后，基于 README 中的上下文信息来响应用户的问题。
